## 📖 About

This repository contains a Jupyter Notebook with solved exercises from the classic textbook:

> **Computer Architecture: A Quantitative Approach**, 7th Edition  
> John L. Hennessy, David A. Patterson, Christos Kozyrakis  
> Morgan Kaufmann, 2024  
> ISBN: 978-0-443-15406-5

The solutions are developed in Python and organized by chapter, covering quantitative analysis, performance evaluation, memory hierarchy, parallelism, and more.

## 🚀 How to Use

### Option 1: Open directly in Google Colab (recommended)

Click the badge at the top of this README or go to:  
[colab.research.google.com](https://colab.research.google.com) → GitHub → paste this repo URL.

### Option 2: Run locally

```bash
git clone https://github.com/VictorAugustoCorrea/computer-architecture-quant-approach-7e.git
cd computer-architecture-quant-approach-7e
jupyter notebook
```

**Requirements:** Python 3.x and Jupyter Notebook (or JupyterLab).
